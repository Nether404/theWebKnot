## Getting Started

### Step 1: Open WebKnot

Visit WebKnot in your web browser. You'll see a beautiful welcome screen with a "Get Started" button.

### Step 2: Click "Get Started"

This launches the wizard that will guide you through creating your prompt.

### Step 3: Follow the Wizard

The wizard has 11 steps. Don't worry - you can go back and change anything at any time!

### Step 4: Generate Your Prompt

At the end, click "Generate Prompt" to create your detailed description.

### Step 5: Use Your Prompt

Copy the prompt and paste it into Bolt.new or WebKnot.ai to build your website!

## Step-by-Step Wizard Guide

### Step 1: Project Setup

**What you'll do:** Tell us about your website idea.

**Fill in:**
- **Project Name**: What's your website called? (e.g., "My Portfolio", "Coffee Shop")
- **Description**: What's it about? (e.g., "A portfolio showcasing my design work")
- **Project Type**: Choose from:
  - Portfolio (showcase your work)
  - E-commerce (sell products)
  - Dashboard (data and analytics)
  - Web App (interactive application)
  - Mobile App (phone/tablet app)
  - Website (general website)
- **Purpose**: Why are you building this? (e.g., "Attract clients", "Sell products")
- **Target Audience**: Who will use it? (e.g., "Design agencies", "Coffee lovers")

**üí° Tip:** Be specific! Instead of "website for business", say "portfolio website to showcase my photography to potential clients".

**ü§ñ AI Help:** Try the "Describe Your Project" feature! Just write naturally like "I want to build a minimalist portfolio with blue colors" and WebKnot will automatically detect your preferences.

3. Click suggested items to apply them

**Example:** Choose Glassmorphism style ‚Üí WebKnot suggests Tech Neon colors and modern components.

**Why it helps:** Ensures all your choices work harmoniously together.

## Troubleshooting

### Common Issues

#### Issue: Can't Click "Get Started"

**Solution:**
1. Refresh your browser (F5 or Ctrl/Cmd + R)
2. Try a different browser (Chrome, Firefox, Safari, Edge)
3. Check if JavaScript is enabled
4. Clear your browser cache

#### Issue: Selections Not Saving

**Solution:**
1. Make sure you're using the same browser
2. Don't use private/incognito mode
3. Check if cookies are enabled
4. Try clicking on another step to trigger save

#### Issue: AI Features Not Working

**Solution:**
1. Complete earlier steps first (AI needs context)
2. Wait a moment for AI to analyze
3. Refresh the page if stuck
4. Check your internet connection

#### Issue: Search Not Finding Components

**Solution:**
1. Try different keywords (e.g., "slide" instead of "carousel")
2. Clear search and browse by category
3. Check spelling
4. Try using tags instead

#### Issue: Prompt Quality Score is Low

**Solution:**
1. Click "Apply Recommendations" button
2. Add missing information:
   - Responsive design
   - Accessibility features
   - Performance optimization
3. Be more specific in descriptions
4. Add more details to project purpose

#### Issue: Can't Copy Prompt

**Solution:**
1. Click the "Copy" button again
2. Try manually selecting text and copying (Ctrl/Cmd + C)
3. Check if clipboard permissions are enabled
4. Try a different browser

#### Issue: Harmony Score Shows Warnings

**Solution:**
1. Read the warning message carefully
2. Click "Auto-Fix" if available
3. Manually adjust conflicting selections
4. Consider AI suggestions in sidebar
